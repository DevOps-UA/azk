## Development Use

1. install node (>= 0.7.0)
    * We recommend to use **nodenv** and **node-build** tools, search at github.
2. clone this repo
    * `$ git clone https://github.com/azukiapp/azk.git` and `$ cd azk`
3. check tests
    * `$ npm test`
4. `$ ./bin/azk help`

5. if you want to install

```sh
$ npm pack
> azk-<VERSION>.tgz
$ npm install azk-<VERSION>.tgz -g
```
